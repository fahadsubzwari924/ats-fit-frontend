How to Use the Button Component

Import: Add ButtonComponent to your component's imports

Basic Usage: <app-button text="Click me" (clicked)="handleClick($event)"></app-button>

With Icons: Use icon="downlaod" iconAlign="left" iconType="svg" or icon="home" iconType="material"

Variants: Choose from primary, secondary, success, danger, warning, info, outline, ghost

Sizes: Available sizes: xs, sm, md (default), lg, xl

States: Use [disabled] and [loading] signals
